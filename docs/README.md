# Website

This website is built using [Next JS](https://nextjs.org/)

### For docs
### Installation

```
$ npm install
```

### Start

```
$ npm start
```

### For web-app
### Installation

```
$ npm install
```

### Local Development

```
$ npm run dev
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

### Build

```
$ npm run build
```
